ES6 提供了新的数据结构 Set。它类似于数组，但是成员的值都是唯一的，没有重复的值。

	注意，在Set内部，两个NaN相等，所以会去除重复的NaN，但是 === 是认为两个 NaN是不等的（NaN===NaN 结果是false）。

	Object和Set在判断是否包括一个key的区别：
	Object：
		if(obj[xxx]){
		    //.....
		}
	Set：

		if(set.has(xxx)){
		    //....
		}


	set的长度使用size进行判定，并会自动去重

	set中NaN等于自身，其余比较相当于 ===
		在set结构中NaN是可以相等的

	Set 对象允许你存储任何类型的唯一值，无论是原始值或者是对象引用。

	set特性：
		成员唯一、无序且不重复；
	[value, value]，键值与键名是一致的（或者说只有键值，没有键名）；
	可以遍历，方法有：add、delete、has、clear、entries、forEach、keys、values
	Set也能用来保存NaN和undefined， 如果有重复的NaN， Set会认为就一个NaN(实际上NaN!=NaN);
