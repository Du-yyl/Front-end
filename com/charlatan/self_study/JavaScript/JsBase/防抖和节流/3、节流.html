2.加入节流后的：【移入div在控制台打印语句】 
	原理：规定一个时间，在这个时间段内的所有事件都会一起被执行
	一般用于：resize，touchmove移动DOM，上拉列表加载数据等

		var input = document.querySelector('input');
        function fn1(callback,time){
            var flag = true;
            return function(){
                if (flag) {
                    flag = false;
                    callback();
                    setTimeout(()=>{
                        flag = true
                    },time)
                }
            }
        }
